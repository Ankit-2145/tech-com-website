Why Tailwind CSS Won (2 minute read)

Tailwind CSS is a ubiquitous frontend styling framework. It became popular because there is no need to context switch to CSS files during development, it makes it easy to copy & paste styles on HTML blocks, and it doesn't depend on other libraries.
